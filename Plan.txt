Config.py
Store file locations, constants, and settings (window size, asset paths).
display.py
Handle pygame display setup, window creation, and drawing functions.
level.py
Manage level data, skateboard angles, and environment (asphalt, cracks, patches).
main.py
Entry point; initialize app, run main loop, handle quitting.
controls.py
Process user input (keyboard/mouse), update skateboard angle or state.
ui.py
Draw UI elements (menus, progress bars, hand indicators) using only horizontal ASCII lines.
